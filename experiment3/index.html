Experiment X - Generative Typing

Description

This experment allwos the user to type out a message and see the characters appear on screen through a system of particles

Technical

Each letter inputed is conveterd to points using the helpful textToPoints() function and then each point is added to an array where the points will be genreated from. The points are then animated using a particle class that is created in a separate file. Each point has a serpate noise value that gives some randomness to the expemrent.
